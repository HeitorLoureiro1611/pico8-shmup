# Pico8-shmup

## O que é pico8?
PICO8 é um "console fantasia" que imita as capacidades gráficas e sonoras limitadas dos sistemas antigos de 8 bits

a proposta de PICO8 é ser desafiador para desenvolvimento de jogos, te entregando um ecossistema que limita a quantidade de interassões que você pode fazer, dessa forma você deve pensar de forma criativa como fazer o seu jogo
